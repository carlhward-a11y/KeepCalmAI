# Illustrations

Illustrations should help learners understand ideas quickly and feel reassured.

They are not decoration.

## Style

KeepCalmAI illustrations should feel:

- Calm
- Friendly
- Modern
- Simple
- Clear
- Inclusive

## Visual approach

Use soft shapes, rounded forms and gentle expressions.

Avoid busy backgrounds, complex scenes or overly technical imagery.

## Colour

Use the KeepCalmAI colour palette.

Colours should support clarity and emotional tone.

Colour must never be the only way information is conveyed.

## Subjects

Useful recurring visual themes:

- AI as a co-pilot
- Small steps
- A map or journey
- Building confidence
- Checking information
- Asking better questions
- Human control

## Accessibility

Every meaningful illustration needs alt text.

Alt text should describe the learning point, not every decorative detail.

## File format

Use SVG where possible.

SVG is scalable, lightweight and easier to adapt for future design changes.
