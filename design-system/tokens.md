# Design Tokens

Design tokens are the shared values used across the KeepCalmAI interface.

They keep the website, lessons and visual assets consistent.

## Colour tokens

- Primary: calm blue
- Secondary: soft teal
- Accent: warm yellow
- Background: off white
- Surface: white
- Text: deep slate
- Muted text: soft grey
- Success: calm green
- Warning: amber
- Error: clear red

Exact colour values will be confirmed during visual prototyping.

## Spacing tokens

Use a simple spacing scale:

- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- xxl: 48px

## Radius tokens

Use rounded corners to create a soft, calm interface.

- Small radius: 8px
- Medium radius: 16px
- Large radius: 24px

## Shadow tokens

Shadows should be subtle.

The interface should feel light, not heavy.

## Motion tokens

Animation should be calm and purposeful.

Use short transitions for progress, completion and navigation.

Avoid distracting or fast movement.
