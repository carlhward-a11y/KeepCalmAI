# Components

KeepCalmAI pages are built from reusable components.

Each component should be simple, accessible and visually consistent.

## Core components

### Hero banner

Used at the start of a journey or major section.

Includes a title, short introduction, optional illustration and primary action.

### Lesson card

Used to show a lesson within a journey.

Includes title, estimated time, difficulty and completion state.

### Step card

Used for each bite-sized learning step.

Includes one idea, one visual if useful and one primary action.

### Try it box

Used when the learner should do something practical.

Should contain clear instructions and, where appropriate, a copyable prompt.

### Tip box

Used for helpful extra guidance.

Should not contain essential information.

### Accessibility note

Used to highlight access features or inclusive ways to complete an activity.

### Confidence checkpoint

Used to ask the learner how confident they feel.

Should offer supportive routes, not judgement.

### Quiz card

Used for short checks of understanding.

Should be brief and encouraging.

### Summary card

Used at the end of a lesson.

Includes key takeaways and the next action.

### Completion card

Used when a learner completes a lesson or journey.

Should reinforce progress and confidence.

## Component rule

Every component must have a clear purpose.

Avoid adding visual elements that do not support learning.
