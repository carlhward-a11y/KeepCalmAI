# KeepCalmAI Design System

The Design System defines the visual and interaction standards used across KeepCalmAI.

Its purpose is to ensure every lesson, page and graphic feels consistent, accessible and reassuring.

## Principles

- Calm before complexity
- Consistency builds confidence
- Accessibility by design
- One idea per screen
- Visuals support understanding
