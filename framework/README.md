# KeepCalmAI Learning Framework

This folder contains the internal standards for KeepCalmAI learning content.

The framework explains how lessons are planned, written, illustrated, reviewed and published.

10. Platform Variations

13. Editorial Workflow

## Core standard

KeepCalmAI content is broken into small, manageable learning steps. Each step focuses on one idea, uses clear language and helps the learner make visible progress.
