# KeepCalmAI Roadmap

This roadmap describes the planned development path for KeepCalmAI.

## v0.1 Foundation

- Create project repository
- Define vision and brand
- Define learning philosophy
- Define audience personas
- Define course architecture
- Define lesson template
- Define writing standards

## v0.2 Design System and Prompt Engine

- Create design tokens
- Define reusable components
- Define illustration standards
- Define accessibility standards
- Create prompt engine
- Create content model

## v0.3 Reference Journey

- Build AI Without Overwhelm
- Add lesson metadata
- Add step-based content
- Add quiz and confidence checkpoints
- Add illustration briefs
- Add accessibility notes

## v0.4 Prototype Website

- Build static prototype
- Add learner journey navigation
- Add progress indicator
- Add quiz interaction
- Add completion screen

## v0.5 Beta Platform

- Add onboarding questions
- Add personalised journey routes
- Add platform-specific variations
- Add searchable content library

## v1.0 Public Launch

- Final accessibility review
- Content review
- Mobile review
- Performance review
- Launch-ready website
