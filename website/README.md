# KeepCalmAI Website

This folder will contain the learner-facing website prototype.

The website should implement the Learning Framework, Design System and content model.

## Prototype goals

- Show a guided learning journey
- Present one step at a time
- Show progress clearly
- Support keyboard navigation
- Use accessible semantic HTML
- Keep visual design calm and uncluttered

- index.html
- styles.css
- scripts.js

- pages/
