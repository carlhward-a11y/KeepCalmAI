# Illustration Generator Prompt

Use this prompt to create consistent KeepCalmAI illustration briefs.

## Prompt

Create an illustration brief for a KeepCalmAI lesson.

Inputs:

- Lesson title:
- Learning point:
- Audience persona:
- Emotion to support:
- Format:
- Aspect ratio:

Requirements:

- Calm, friendly and simple visual style
- Soft rounded shapes
- Minimal background detail
- Clear learning purpose
- Inclusive representation
- No visual clutter
- No text embedded in the image unless essential
- Suitable for SVG or vector-style production

Output:

- Illustration description
- Suggested composition
- Colour notes
- Accessibility notes
- Alt text
- Optional animation idea

The illustration must support understanding. It should not be decorative filler.
