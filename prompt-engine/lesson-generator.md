## Prompt

Create a KeepCalmAI lesson using the Learning Framework and Design System.

Inputs:

- Journey:

Requirements:

- Use UK English
- Keep the tone calm, practical and reassuring
- Break the lesson into 3 to 6 short steps
- Each step should focus on one idea
- Include a suggested visual for each key concept
- Include alt text for each meaningful visual
- Include one Try it activity
- Include one confidence checkpoint
- Include a short summary
- Include metadata at the top

Output format:

Markdown.

Do not create a long article. Create a guided learning experience.
