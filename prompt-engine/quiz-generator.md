# Quiz Generator Prompt

Use this prompt to create short KeepCalmAI knowledge checks.

## Prompt

Create a short quiz for a KeepCalmAI lesson.

Inputs:

- Lesson title:
- Learning objective:
- Audience persona:
- Difficulty:

Requirements:

- Use supportive language
- Avoid trick questions
- Keep questions short
- Use plain English
- Include 2 to 4 questions only
- Provide feedback for each answer
- Reinforce confidence

Question types:

- Multiple choice
- True or false
- Confidence checkpoint
- Practical scenario

Output:

Markdown with questions, answer options, correct answer and feedback.
