# KeepCalmAI Prompt Engine

The Prompt Engine contains reusable prompts for generating consistent KeepCalmAI content and assets.

Each prompt should follow the Learning Framework and Design System.

## Prompt types

- Journey generator
- Lesson generator
- Step generator
- Quiz generator
- Illustration generator
- HTML generator
- Accessibility reviewer
- Editorial reviewer
- Rewrite and simplify

## Standard inputs

Prompts should usually ask for:

- Audience persona
- Collection
- Journey name
- Lesson objective
- AI platform
- Difficulty level
- Estimated time
- Accessibility requirements
- Output format

## Output principle

Generated content should be calm, practical, accessible and confidence-building.
