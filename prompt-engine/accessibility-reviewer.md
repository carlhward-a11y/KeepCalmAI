# Accessibility Reviewer Prompt

Use this prompt to review KeepCalmAI content before publication.

## Prompt

Review the supplied KeepCalmAI lesson for accessibility, clarity and cognitive load.

Check:

- Plain English
- Clear headings
- Short paragraphs
- One idea per step
- One main action per screen
- Meaningful alt text
- Keyboard-friendly interaction wording
- Screen reader clarity
- Colour is not used as the only cue
- No unnecessary time pressure
- No patronising language
- No jargon without explanation

Output:

1. Summary judgement
2. Strengths
3. Issues to fix
4. Suggested rewrite where needed
5. Publication recommendation

Recommendation options:

- Ready
- Ready with minor edits
- Needs revision
- Not suitable yet
