# Prompt basics

A prompt is the question or instruction you give to an AI tool.

Good prompts are usually clear and specific.

## A simple pattern

Say the task.

Add a little context.

Say the style you want.

## Example

Help me write a short note. Make it friendly and clear. It is for a neighbour.

## Try this

Write one short prompt about a task you would like help with.

## Key takeaway

You can change or improve a prompt after you see the first answer.
