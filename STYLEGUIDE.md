# KeepCalmAI Style Guide

This guide defines the writing and presentation standards for all KeepCalmAI content.

## Language

Use UK English.

Write clearly and calmly.

Avoid jargon unless it is explained.

## Tone

KeepCalmAI should sound:

- Calm
- Practical
- Reassuring
- Respectful
- Human

## Structure

Use short sections and clear headings.

Avoid long pages of continuous text.

Each learning step should focus on one idea.

## Words to use carefully

Avoid words that may make learners feel judged or behind, including:

- easy
- obvious
- simple
- just
- master
- hack

Use supportive alternatives such as:

- try this
- one way to do this
- this is normal
- you are still in control

## Accessibility

Content should be usable with screen readers, magnification, keyboard access and cognitive support needs.

Include meaningful alt text for meaningful visuals.

Do not rely on colour alone.
