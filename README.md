# KeepCalmAI

Calm Today. Confident Tomorrow.

KeepCalmAI is a learning platform for people who want to understand and use AI without jargon, pressure or information overload.

This repository contains the framework, content structure, design standards and future website assets for the KeepCalmAI project.

## Repository structure

```text
framework/   Internal learning framework and content standards
content/     Bite-sized learning journeys and reusable lesson blocks
graphics/    Illustrations, diagrams, icons and brand assets
prompts/     Reusable prompts for content and visual generation
website/     Learner-facing website and platform files
assets/      Shared supporting files
```

## Guiding principle

Every learner should finish each session feeling more confident than when they started.
